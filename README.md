# ci-sparks
A CLI tool for generating [Cinder](https://libcinder.org/) cmake projects written in go.

### usage:
1. set the default cinder path in the `config.json`
2. run ./ci-sparks --name "MyCinderProject" --dest "/project/destination/folder"
